# 酒店图片制作处理平台 - MVP 需求文档

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | 酒店图片工坊 |
| **产品定位** | 为酒店从业者提供快速、免费的图片在线处理工具 |
| **核心价值** | 无需安装软件，浏览器在线处理酒店图片，支持 AI 增强 |
| **目标用户** | 酒店民宿业主、OTA 运营人员、酒店摄影师 |

---

## 2. MVP 功能范围

### 2.1 必须实现（MVP）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **图片上传** | 支持拖拽上传，单张/批量（≤10张），支持 jpg/png/webp | P0 |
| **基础裁剪** | 自由裁剪、预设比例（16:9、4:3、1:1、2:3） | P0 |
| **图片旋转** | 90° 旋转、水平/垂直翻转 | P0 |
| **尺寸调整** | 输入宽高或按比例缩放 | P0 |
| **画质压缩** | 调整质量参数，实时预览大小 | P0 |
| **格式转换** | jpg ↔ png ↔ webp 互转 | P0 |
| **AI 画质增强** | 调用 MiniMax API 提升模糊图片清晰度 | P0 |
| **图片下载** | 单张下载、批量打包下载 | P0 |

### 2.2 后续迭代

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **AI 背景移除** | 自动识别并移除图片背景 | P1 |
| **AI 扩图** | 智能扩展图片边缘 | P1 |
| **预设模板** | 携程/美团/飞猪等平台尺寸模板 | P1 |
| **水印添加** | 文字/图片水印 | P2 |
| **批量处理** | 一次处理多张图片 | P2 |

## 4. 技术架构

### 4.1 技术栈

| 层级 | 方案 |
|------|------|
| **前端框架** | React + TypeScript + Vite |
| **UI 组件** | Tailwind CSS + shadcn/ui |
| **后端服务** | Cloudflare Workers |
| **图片处理** | @aspect-ratio/sharp (Workers 兼容) |
| **AI 能力** | MiniMax API |
| **部署平台** | Cloudflare Pages + Workers |

### 4.2 架构图

```
                    ┌─────────────────┐
                    │  Cloudflare     │
                    │     CDN         │
                    └────────┬────────┘
                             │
        ┌────────────────────┼────────────────────┐
        │                    │                    │
        ▼                    ▼                    ▼
┌───────────────┐   ┌───────────────┐   ┌───────────────┐
│  Pages 静态   │   │   Workers    │   │   MiniMax    │

                            │
                     ┌──────┴──────┐
                     │  内存处理    │

                     └─────────────┘
```

### 4.3 限制说明

| 限制项 | 值 | 说明 |
|--------|-----|------|
| 单次上传大小 | ≤10MB | Cloudflare 限制 |
| 并发处理 | ≤3 张 | 内存限制 |
| 请求超时 | 30s | Worker 限制 |
| 图像尺寸 | ≤4000×4000 | 内存限制 |

## 5. API 设计

### 5.1 接口列表

| 方法 | 路径 | 描述 |
|------|------|------|
| `POST` | `/api/upload` | 上传图片（Base64） |
| `POST` | `/api/crop` | 裁剪图片 |
| `POST` | `/api/resize` | 调整尺寸 |
| `POST` | `/api/rotate` | 旋转/翻转 |
| `POST` | `/api/compress` | 压缩图片 |
| `POST` | `/api/convert` | 格式转换 |
| `POST` | `/api/ai/enhance` | AI 画质增强 |
| `GET` | `/api/health` | 健康检查 |

### 5.2 请求/响应格式

**请求示例：裁剪图片**
```json
{
  "image": "data:image/jpeg;base64,/9j/4AAQ...",
  "x": 100,
  "y": 100,
  "width": 800,
  "height": 600
}
```

**响应示例**
```json
{
  "success": true,
  "result": "data:image/jpeg;base64,/9j/4AAQ...",
  "size": 245678,
  "width": 800,
  "height": 600
}
```

## 8. 验收标准

### 8.1 功能验收

- [ ] 支持 jpg/png/webp 格式上传
- [ ] 裁剪功能正常，预设比例可用
- [ ] 旋转翻转功能正常
- [ ] 压缩可预览文件大小变化
- [ ] 格式转换正确
- [ ] AI 增强有可见效果
- [ ] 下载功能正常

### 8.2 性能验收

- [ ] 页面首屏加载 < 2s
- [ ] 基础处理响应 < 3s
- [ ] AI 处理响应 < 10s

### 8.3 异常处理

- [ ] 大文件上传有提示
- [ ] 不支持的格式有报错
- [ ] AI API 超时有重试
